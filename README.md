go-itertools
============
Implemented Python's `itertools` in Golang.

This is just a practice.


Python's itertools functions
============================
Implemented:
 - accumulate
 - chain
 - chain.from_iterable
 - consume
 - cycle
 - ncycle
 - nth
 - repeat
 - repeatfunc
 - take

Not implemented:
 - combinations
 - combinations_with_replacement
 - compress
 - dotproduct
 - dropWhile
 - filterFalse
 - flatten
 - group_by
 - grouper
 - iter_except
 - padnone
 - pairwise
 - partition
 - permutations
 - powerset
 - product
 - quantify
 - random_product
 - roundrobin
 - slice
 - startmap
 - tabulate
 - takeWhile
 - tee
 - unique_ever_seen
 - unique_just_seen
 - zip_longest
